# Privacy Policy for XP Quest Bot

**Last Updated:** April 2, 2026

## 1. Information We Collect

The XP Quest Bot collects and stores the following information:

- **User Identification**: Telegram user ID, username, and first name
- **Activity Data**: XP points earned, total XP, spent XP
- **Profile Data**: Selected departments, language preference, user role
- **Submission Data**: Task submissions, proof attachments, ideas shared
- **Timestamps**: When users registered, when they completed tasks

## 2. How We Use Your Information

We use collected data to:

- Display user profiles and leaderboard rankings
- Track task completion and reward XP
- Manage department assignments and permissions
- Process task submissions and approve rewards
- Provide personalized user experience
- Generate statistics and analytics for administrators

## 3. Data Storage and Security

- Data is stored in a local SQLite database
- Database file: `bot_data.db`
- No data is shared with third parties
- Access is restricted to authorized bot administrators only
- We do not sell or trade personal information

## 4. User Rights

You have the right to:

- **Access**: Request all data we hold about you
- **Deletion**: Request deletion of your account and associated data
- **Correction**: Request corrections to your profile information

To exercise these rights, contact the bot administrators directly via the `/help` command.

## 5. Data Retention

- User data is retained as long as your account exists
- Deleted accounts have their data removed from the database
- Administrators may archive historical data for statistics

## 6. Admin Features

Administrators can:

- View all user statistics and rankings
- Review submitted tasks and approve/deny rewards
- Manage user accounts and reset data if needed
- Access analytics about bot usage

## 7. Changes to This Policy

We may update this privacy policy as the bot evolves. Changes will be announced through the bot.

## 8. Contact

For privacy concerns or to request your data, use the `/help` command in the bot to contact administrators.

---

**By using XP Quest Bot, you agree to the collection and use of information as outlined in this policy.**
