# 📖 Гайд користувача — Як використовувати Валєру

Детальний, крок за кроком гайд як користуватися ботом для звичайних членів АТУР.

**Час на прочитання:** 15 хвилин  
**Рівень складності:** Новачок  
**Потрібні знання:** Базовий Telegram

---

## 🚀 Швидкий старт (5 хвилин для поспішних)

Якщо тебе не цікавлять деталі — ось мініатюрна версія:

1. **Запустити бота** → Знайди @xp_bot_atur в Telegram, натисни Start
2. **Вибрати департамент** → Натисни кнопку свого департаменту
3. **Подати ідею** → Натисни "🎯 Подати ідею", напиши опис
4. **Чекати** → Супервайзер затвердить протягом 1-3 днів
5. **Святкувати** → Твоя ідея в системі, ти в лідерборді! 🎉

**Детальний гайд продовжується далі ↓**

### Что каждая кнопка делает:

| Кнопка | Що це | Для чого | Час |
|---|---|---|---|
| **🎯 Подати ідею** | Форма для написання ідеї | Пропонувати нові ідеї спільноті | 5-10 хв |
| **📊 Лідерборд** | Таблиця активності членів | Видеть хто найбільше працює | 2-5 хв |

| **📚 Документація** | Цей гайд | Допомога й FAQ | 15 хв |
| **❓ Допомога** | Список команд | Швидко дізнатися команди | 1 хв |

## Розділ 3️⃣: Подання Ідеї (Найважніше! 🎯)

Це найважливіша функція Валєри. Ось як подати ідею прямо з Вам.

### 3.1 Натиснути "🎯 Подати ідею"

На головному меню натисни кнопку:

```
┌─────────────────────────
│ 🎯 Подати ідею      ← натисни тут
│ 📊 Лідерборд
│ ⭐ Мій профіль
│ ⚙️  Налаштування
│ 📚 Документація
│ ❓ Допомога
└─────────────────────────
```

### 3.2 Крок 1: Назва Ідеї

Бот запитує:

```
💡 Подання ідеї

Дай назву своій ідеї:
[__________________________________]  ← напиши назву

(Коротко й ясно. Примеры: "Турпоїздка в гори", "Веб-сайт АТУР")
```

**Що писати:**
- ✅ "Організувати шахматный турнір"
- ✅ "Запустити YouTube канал про мистецтво"
- ✅ "Допомогти українцям з пошуком роботи"
- ❌ "Щось крутое" — занадто розпливчасто
- ❌ "idddddddddd" — не має сенсу

**Напиши назву й натисни надіслати.**

### 3.3 Крок 2: Деталі Ідеї

Бот запитує більше информации:

```
📝 Деталі ідеї

Опиши детальніше — що, як, коли, чому?

[___________________________________
 ___________________________________
 ___________________________________
 ___________________________________]  ← напиши опис

Приклад:
"Організувати шахматный турнір АТУР.
Коли: травня 2024, вихідні
Де: в Палаці культури
Як: 8 учнів, 3 раунди, призи для們者
Скільки коштує: 50 лей для учасників"
```

**Що тут писати:**
1. **ЧТО** — що саме ти пропонуєш?
2. **КОЛИ** — коли це мало робиться?
3. **ДЕ** — де це буде (місто, адреса)?
4. **КИМ** — хто буде робити (сам, команда)?
5. **ВАРТІСТЬ** — скільки приблизно коштує (якщо щось)?
6. **РЕЗУЛЬТАТ** — який результат очікуєш?

**Приклади детальних описів:**

#### Приклад 1: Спортивна Подія
```
Організувати футбольний матч АТУР vs інші асоціації.
Коли: 10 червня 2024, 18:00
Де: стадіон імені Стефана Великого, Кишинів
Учасники: 11 від АТУР + 11 від інших
Спорядження: м'ячі, світлильники, медики
Бюджет: 500 лей (оренда, світло, медики)
Результат: Розвиток спорту в асоціації, команднього духу.
```

#### Приклад 2: Культурна Подія
```
Виставка українського мистецтва в Кишиневі.
Коли: з 1 по 15 липня 2024
Де: Художній музей (центр міста)
Робітниці: я, Марія, Петро (3 особи)
Експозиція: 50 картин від членів АТУР
Рекламо: Facebook, Instagram, газети
Вартість: 1000 лей (оренда, каталоги, афіші)
Результат: Показати культуру АТУР суспільству.
```

#### Приклад 3: Розвивальна Подія
```
Безплатні курси програмування для дітей українців.
Коли: юні 2024, 2 рази в неділю по 2 години
Де: онлайн на Zoom
Дітям: 10-16 років, дітей членів АТУР
Програма: Python основи, 8 занять
Вчитель: я + 1-2 від асоціації
Матеріали: підручники, прості завдання
Результат: 20-30 дітей навчать кодити, розвивають навички.
```

**Натисни надіслати коли закінчив писати.**

### 3.4 Крок 3: Анонімність

Бот запитає:

```
🕵️ Хочеш поділитися ім'ям чи залишитися анонімним?

[✅ Поділитися ім'ям]  ← бот покаже твоє ім'я
                       всім членам АТУР

[🕵️ Анонімно]          ← твоє ім'я не видно
                       в системі, тільки судовий бот
```

**Плюси кожного варіанту:**

| Вибір | Плюси | Мінуси |
|---|---|---|
| **Поділитися ім'ям** | Всі знають що це твоя ідея, видно вклад | Менш приватно |
| **Анонімно** | Ніхто не знає хто подав, повна приватність | Числе не видно твій вклад в лідерборді |

**⚠️ ВАЖНО:** Якщо обереш анонімну:
- Твоя ідея буде в системі, але як безумовное/невідомого автора
- Супервайзер все одно бачить хто подав
- **В лідерборді** твоя ідея не лічится! (для анонима)
- **Рекомендуємо** поділитися ім'ям, щоб отримати визнання

**Натисни свій вибір.**

### 3.5 Крок 4: Підтвердження

```
📤 Ідея відправлена!

Твоя ідея:
"Организувати шахматный турнір"

✅ Статусу: Нова (очікує розгляду)

📬 Ти отримаєш сповіщення про результат

[← Назад в меню]

**ВСЕ! ТВоя ідея в системі!** 

Тепер залишається чекати. Супервайзер переглядає ідеї в своєму департаменті протягом 1-3 днів.

### 3.6 Отримати Результат

Через 1-3 дні ти отримаєш сповіщення:

```

**Якщо відхилена:** ❌
```

"Организувати шахматний турнір"
Супервайзер каже: "Ідея цікава, але потребує більше деталей. 
Скільки бюджету? Де саме? Адреса уточни."

Ти можеш:
1️⃣ Поправити й подати нову версію
2️⃣ Обговорити з супервайзером в чаті

[Обговорити] [Подати заново] [Назад]
```

## 🆘 Проблемы й Розв'язання

### ❓ Що робити якщо...

#### ...мья ідея відхилена? ❌

```
Ти отримав повідомлення:
❌ Ваша ідея не затверджена

Супервайзер писал: "Дата не ясна. Яке число міцнах саме?"

ЧТО РОБИТИ:
1. Підправи описання (додай дату, адресу, детали)
2. Натисни [Подати заново]
3. Напиши покращену версію
4. Супервайзер переглядує ще раз
```

#### ...я не знаю до якого департаменту подати ідею? 🤔

```
РІШЕННЯ:
1. Подай ідею до найбільш підходящого (IT для програмування, Культура для мистецтва)
2. Супервайзер може перекинути в інший департамент якщо потрібно
3. Після затвердження можна змінити

ПРИКЛАД:
Ідея: "Запустити веб-сайт АТУР"
→ Подай в IT (вони займаються технологієми)
```

#### ...я не отримую сповіщення? 🔔

```
ПЕРЕВІР:
1. ⚙️ Налаштування → 📬 Сповіщення → ✅ Увімкнені?
2. Дай дозвіл Telegram на сповіщення (OS налаштування)
3. Напиши адміну якщо все корректно

ТИМЧАС:
Можеш сам дивитися свої ідеї:
/idea → [Мої ідеї] → бачиш весь статус
```

#### ...я забув мій пароль? 🔐

```
ЦЕ НЕ ПРОБЛЕМА!
Валєра не використовує пароль!
Ти входиш через Telegram ID (телефон).

ЩО РОБИТ:
1. Натисни [/start] в ботові
2. Твій Telegram автоматично підтасте
3. Ви знахідиді свий аккаунт за ID
4. Готово — уві в ботові!
```

#### ...мая ідея на українській, а мова в боті румунська? 🌍

```
РІШЕНЄІЯ:
1. Натисни ⚙️ Налаштування
2. 🌍 Змінити мову → [🇺🇦 Українська]
3. Все переходит на українською!

ПРИМІТКА: Мова в ботові — це мова інтерфейсу (кнопки, меню).
Текст ідеї якy ТИ ПИШЕШ — будь-яка мова OK.
```

#### ...я хочу змінити ім'я? 📝

```
ІНСТРУКЦІЯ:
1. Напиши адміністратору Telegram
2. Скажи: "Хочу змінити ім'я в Валєрі"
3. Адмін допоможет поміняти

ПРИМІТКА: Ім'я змінюється в системі, старі ідеї залишаються при тобі.
```

#### ...я хочу видалити мій аккаунт? 🗑️

```
ДЕ ПОЧАТ:
1. Напиши розробнику або адміну
2. Скажи: "Хочу видалити профіль"
3. Вони видалять все твої дані

ВАРТО ПОМІТИТИ: Після видалення:
- Твої ідеї видаляються (або переноситься як "Анонімна ідея")
- Твої рейтинги видаляються
- Це не можна скасувати!
```
